# SMITAssig2
# Python Assignment - Basic Programs

This repository contains solutions for a Python assignment covering **basic programming problems**.  
The assignment includes 20 different programs that take user input, perform calculations, and display results.

---
**List of Programs**

| #  | Program Name | Description |
|----|-------------|-------------|
| 1  | Area of a Rectangle | Calculates area using `length * width` |
| 2  | Circumference of a Circle | Calculates circumference using `2 * π * r` |
| 3  | Simple Interest | Calculates simple interest using `P * R * T` |
| 4  | Speed of an Object | Calculates speed using `distance / time` |
| 5  | BMI Calculator | Calculates Body Mass Index using `weight / height^2` |
| 6  | Force Using Newton's Second Law | Calculates force using `F = m * a` |
| 7  | Compound Interest | Calculates compound interest using `A = P * (1 + r/n)^(n*t)` |
| 8  | Perimeter of a Triangle | Calculates perimeter using `a + b + c` |
| 9  | Volume of a Sphere | Calculates volume using `V = 4/3 * π * r^3` |
| 10 | Kinetic Energy | Calculates KE using `0.5 * m * v^2` |
| 11 | Quadratic Equation Roots | Finds roots using `(-b ± sqrt(b^2 - 4ac)) / 2a` |
| 12 | Temperature Conversion | Converts Celsius to Fahrenheit using `F = C * 9/5 + 32` |
| 13 | Gravitational Force | Calculates gravitational force using `F = G * m1 * m2 / r^2` |

| 17 | Perimeter of a Circle (Circumference) | Calculates circumference using `C = 2 * π * r` |
